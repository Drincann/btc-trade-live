# BTC Trade Live

Realtime Bitcoin trade flow dashboard for GitHub Pages.

The app connects to Binance Spot WebSocket trade streams and renders a live price chart, order-flow metrics, and animated trade rows.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8766
```

Then visit `http://127.0.0.1:8766/`.

## Deploy

This repository is designed for GitHub Pages from the repository root.
